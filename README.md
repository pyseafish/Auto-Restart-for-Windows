## 📖 简介

**自动重启工具** 是一款运行在 Windows 系统托盘中的后台实用程序。它能够按照您设定的周期（默认72小时）**自动重启电脑**，并在重启登录后**自动启动您指定的多个应用程序**，同时**确保自身在开机后自动运行**，形成完整的自动化闭环。

无论您是运维人员需要定时重启测试机，还是普通用户希望每天开工时自动打开浏览器、邮件客户端、IDE，这款工具都能安静地为您代劳。

---

## ✨ 功能特色

- **⏱️ 完全可定制的重启间隔**  
  支持以“小时”为单位设置重启周期（1\~720小时，即1小时\~30天），默认72小时（3天）。

- **🚀 重启后自动启动多个程序**  
  支持通过图形界面**一次性添加多个可执行文件**（.exe），并可自由调整启动顺序、删除不需要的程序。

- **🖥️ 开机自启动（可选）**  
  提供简洁的复选框，用户可随时启用/禁用软件随系统登录自动运行，无强制行为。

- **🔕 零干扰设计**  
  软件启动后**完全隐藏**，仅显示托盘图标，不占用任务栏，不弹出任何窗口。所有交互均通过托盘右键菜单完成。

- **🛡️ 单实例保护**  
  利用互斥体（`Mutex`）确保系统中只有一个实例运行，避免重复启动。

- **🧹 干净的状态管理**  
  所有配置（重启间隔、程序列表）均存储在**当前用户注册表**中，无需管理员权限，卸载时可通过脚本一键清理。

- **操作系统**：Windows 7 SP1 / 8 / 10 / 11（32位/64位均可）

## 📖 使用指南

### 第一次启动
- 双击 `AutoRebootTool.exe`，程序将**自动最小化到系统托盘**（任务栏右侧）。
- 托盘图标为默认的灰色方块（可自定义），鼠标悬停显示“自动重启工具”。
- **右键点击图标**弹出菜单。

### 配置软件
1. 右键托盘图标 → **“设置”**。
2. **重启间隔**：输入您希望的小时数（例如 `72` 表示3天）。
3. **添加程序**：点击“添加程序...”，可**按住 Ctrl 或 Shift 多选**多个 `.exe` 文件，一次性加入列表。
4. **调整顺序**：选中程序后，点击“上移”/“下移”可调整启动顺序（先启动列表顶部的程序）。
5. **删除**：可多选后点击“删除选中”。
6. **开机自启**：若希望本软件随系统启动，**勾选**“开机自动运行本软件”。
7. 点击 **“保存”**，配置立即生效。

### 立即重启
- 右键托盘图标 → **“立即重启”**，系统将在 **0秒后** 强制重启。
- 重启登录后，您添加的所有程序将**按顺序自动启动**，同时本软件（若勾选了开机自启）也会自动运行。

### 退出软件
- 右键托盘图标 → **“退出”**，软件完全终止，不再后台运行。
